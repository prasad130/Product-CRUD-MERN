# Product CRUD Application (MERN Stack)

## Project Overview

This is a Product Management Application built using the MERN Stack.

The application allows users to:

- Add Products
- View Products
- Update Products
- Delete Products

The frontend is built with React.js and Axios.

The backend is built with Node.js, Express.js, MongoDB, and Mongoose.

---

## Technologies Used

### Frontend

- React.js
- Axios
- CSS

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

---

## Features

- Create Product
- Display Products
- Update Product
- Delete Product
- Form Validation
- Delete Confirmation
- Responsive UI
